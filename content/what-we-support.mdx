# What we support

The Paradym Wallet was built to support the latest developments in the European Digital Identity ecosytem.


|         | Supports |   Notes |
| :------------ | :---------: | ----------: |
| EUDI Credential formats        |    SD-JWT VC    |  |
|         |   ISO 18013-5 mDoc    |  |
| OpenID4VCI        |    1.0    |  |
| OpenID4VCP     |    1.0     |     |
| Authorization code flows |   Presentation during issuance        |     |
| |  Sign in           |     |
| |  Transaction code           |     |
| Deferred Issuance|      ✔️       |     |
| Digital Credentials API|     ✔️        |  Only using OpenID4VP on Android   |
